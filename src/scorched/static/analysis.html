Data Dictionary
Every morning, Phase 0 fetches data from 11 sources. Phase 1 pre-filters to the top ~25 symbols, then sends a structured research context to Claude for analysis.
Alpaca Data API API KEY
Price Bars (1 year, IEX feed)
Daily OHLCV for all symbols. Used for: current price, returns, 52-week range, all technical indicators. Replaced yfinance for price data (more reliable, no scraping).
Snapshots
Real-time latest trade price, previous close, daily OHLV. Used for pre-market gap detection and intraday monitoring.
News
Up to 5 articles per symbol: headline + summary + source. Replaced Polygon.io for news (free with Alpaca account, no extra key).
Screener
Most active stocks by volume and top gainers/losers. Supplements the momentum screener with market-wide activity data.
LLM use: Primary data backbone for prices and news. Claude sees price action, momentum, and catalyst headlines per symbol. Batch API — fast and reliable.

Fundamentals
P/E ratio, forward P/E, EPS, market cap, short ratio, short % of float.
Earnings History
Last 4 quarters: EPS estimate vs actual, surprise %, beat/miss/inline verdict.
Options Chains
Put/call ratio, ATM implied volatility, 30-day implied move. Fetched only for Call 1 candidates.
Insider Purchases
Recent insider transactions from yfinance. Supplements EDGAR Form 4 data.
Index Prices (^GSPC, ^VIX, etc.)
Alpaca only covers equities/ETFs. yfinance still used for index symbols like S&P 500, VIX, and DJI.
LLM use: Fundamentals and earnings backbone. Claude cross-references valuation metrics with momentum signals. Also sole source for index prices and options data.
Technical Analysis (from Alpaca bars) COMPUTED
MACD
12/26/9 EMA crossover. Signal: BULLISH (histogram positive + rising), BEARISH, or NEUTRAL.
Bollinger Bands
20-day SMA ± 2 std dev. %B position: OVERBOUGHT (>1.0), OVERSOLD (<0.0), NEUTRAL.
MA Crossover
50-day vs 200-day SMA. Detects golden cross, death cross, or relative position (above both, below both, between).
ATR (14-day)
Average True Range in dollars and % of price. Guides volatility-adjusted stop-loss placement.
Volume Profile
20-day avg volume, relative volume (latest/avg). HIGH (>1.5x), LOW (<0.5x), NORMAL.
Support / Resistance
Simple 20-day high/low range as resistance/support levels.
LLM use: Pre-labeled signals (BULLISH/BEARISH) so Claude can quickly triage. ATR informs stop-loss recommendations.
Sector Relative Strength COMPUTED
Relative Strength
Stock's 5-day return minus its sector ETF's 5-day return (via Alpaca batch bars). Positive = outperforming sector. Covers 11 sector ETFs (XLK, XLF, XLV, XLE, XLI, XLC, XLY, XLP, XLB, XLU, XLRE).
LLM use: Distinguishes true momentum from sector beta. A stock up 3% in a sector up 5% is actually weak.
FRED (Federal Reserve) API KEY
Fed Funds Rate
Current rate + change. Signals monetary policy direction.
Yield Curve (10Y-2Y)
Spread between 10-year and 2-year Treasury yields. Inversion is a recession signal.
CPI YoY
Consumer Price Index year-over-year change. Inflation gauge.
Unemployment Rate
Current + previous reading. Labor market health.
HY Credit Spread
High-yield corporate bond spread over treasuries. Widening = risk aversion.
Consumer Sentiment, PCE, Industrial Production, Retail Sales
Additional macro indicators with current value, previous value, and change.
LLM use: Macro overlay. Claude classifies the environment as bullish/neutral/bearish before evaluating individual stocks. 3+ bearish macro indicators = "no new buys."
Twelvedata API KEY
RSI (14-day) — Full Watchlist
Relative Strength Index for ALL research symbols (800 calls/day free tier). Broader coverage than Alpha Vantage. OVERBOUGHT (>70), OVERSOLD (<30), NEUTRAL.
LLM use: Momentum confirmation across the full universe. Alpha Vantage covers screener picks only; Twelvedata provides RSI for everything.
Finnhub API KEY
Analyst Consensus
Strong Buy / Buy / Hold / Sell / Strong Sell counts. Bullish % computed. Used to gauge Wall Street sentiment.
LLM use: Confirmation/contradiction signal. Claude cross-references analyst consensus with its own technical and catalyst analysis.
SEC EDGAR FREE
Form 4 Filings
Count of insider transaction filings in the last 30 days. The EDGAR submissions API doesn't distinguish buy vs sell — reported as "filing count, type unknown."
LLM use: Insider activity flag. High Form 4 activity prompts Claude to investigate further. Falls back to yfinance insider data if EDGAR fails.
Economic Calendar (FRED) API KEY
Upcoming Releases
Tracks CPI, Jobs (NFP), FOMC, GDP, PPI, PCE, retail sales, housing starts, consumer confidence, industrial production. Warns Claude about same-day releases that could move markets.
LLM use: Event risk overlay. Claude factors in upcoming data releases when sizing positions and setting stops. Same-day releases may trigger caution.
Alpha Vantage API KEY
RSI (14-day) — Screener Picks Only
RSI for momentum screener picks only (max ~20 symbols). Free tier: 25 calls/day. Twelvedata provides broader RSI coverage for the full watchlist.
LLM use: Secondary RSI source for screener candidates. Twelvedata covers the full watchlist; Alpha Vantage fills in for any gaps.
Momentum Screener COMPUTED
Top 20 S&P 500 Movers
Scans ~430 S&P 500 stocks (excluding watchlist) via batch download. Filters: price > 20d MA, avg volume > 1M. Ranked by 5-day momentum. Adds fresh ideas beyond the fixed watchlist.
LLM use: Expands the candidate universe. These symbols are added to the research context so Claude can evaluate emerging momentum plays.
